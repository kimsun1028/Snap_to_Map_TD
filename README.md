# Snap_to_Map_TD

## 프로젝트 소개
OpenCV로 사진 속 테두리를 추출해 디펜스 맵을 생성하고,
원본 사진 위에서 게임이 진행되는 이미지 기반 디펜스 게임입니다.

## 핵심 파이프라인
1. 맵 생성 (OpenCV, Python)
   사용자가 촬영한 사진에서 이동 경로(Path)를 추출합니다.
2. 데이터 전달
   OpenCV에서 계산한 좌표 리스트를 게임 엔진(Unity)으로 전달합니다.
3. 게임 로직 구현 (Unity, C#)
   전달받은 좌표를 기반으로 맵과 디펜스 게임 로직을 구현합니다.
